# 学科分类参考（Discipline Reference）

本文件定义本 skill 卡片 `deck` / `category` 的学科归类标准。**生成新卡或迁移旧卡时，按此判定归属。**

- `deck` = **一级学科（习惯名）** —— 用通行学科名（如「金融学」「心理学」「音乐」），不用抽象层（不写「应用经济学」）。
- `category` = **二级学科 / 细分方向** —— 中文、**单名、无 `/`**。

## 体系

采用中国《研究生教育学科专业目录》三级结构：**学科门类 → 一级学科 → 二级学科**。

> 说明：2011 年起国家目录只统一规定到**一级学科**，二级学科改由培养单位自设、不再全国统一列。因此本 ref 的二级学科是**常用参照**，模型可据自身学科知识补充更贴切的细分方向。**以国务院学位委员会现行目录为准。**

## 13 学科门类

哲学 · 经济学 · 法学 · 教育学 · 文学 · 历史学 · 理学 · 工学 · 农学 · 医学 · 军事学 · 管理学 · 艺术学
（2020 年起另设「交叉学科」门类。）

## 一级学科锚点（按门类，节选常用）

| 门类 | 常用一级学科（= deck 候选） |
|---|---|
| 哲学 | 哲学（二级：中国哲学、外国哲学、伦理学、逻辑学、美学、宗教学） |
| 经济学 | 理论经济学、应用经济学（**习惯名：金融学 / 国际贸易 / 财政学 / 产业经济学**） |
| 法学 | 法学、政治学、社会学、马克思主义理论、民族学 |
| 教育学 | 教育学、**心理学**、体育学 |
| 文学 | 中国语言文学、外国语言文学、新闻传播学 |
| 历史学 | 中国史、世界史、考古学 |
| 理学 | **数学**、统计学、物理学、化学、生物学、地理学、系统科学 |
| 工学 | 计算机科学与技术、软件工程、电子信息、控制科学与工程、机械工程…（共 38，余以官方为准） |
| 管理学 | 管理科学与工程、工商管理、公共管理、图书情报与档案管理 |
| 艺术学 | 音乐与舞蹈学（习惯名「音乐」）、美术学、设计学、戏剧与影视学、艺术学理论 |
| 医学 / 农学 / 军事学 | （按需，以官方为准） |

> 置信度：门类与主流一级学科 [KNOWN·HIGH]；工学/医学等门类的全部一级学科未穷举 [LOW]，以官方目录为准。

## 归类原则

1. **先判学术归属**：内容属于哪个学术领域 → 取最贴切的**一级学科（习惯名）**作 `deck`。
2. **实操 / 民间 / 交叉内容**：归到**最接近的学术上位学科**（见下表），不要因为「不是学术」就拒绝归类。
3. `category` 取该 deck 下的**二级学科或具体细分方向**，中文、单名、无 `/`。
4. **拿不准时**：`deck` 宁可取较宽的一级学科，把具体性体现在 `category`。

## 实操 / 民间内容归类示例

| 内容 | → deck（一级学科） | → category（二级方向） |
|---|---|---|
| 实操交易 / 技术分析 / 宏观 / 衍生品 / ETF / 债券 | 金融学 | 技术分析 · 宏观流动性 · 衍生品 · 资产配置 … |
| 德州扑克 | 数学 | 博弈论 |
| 把妹 / 吸引力 / 社交动力学 | 心理学 | 社会心理学 |
| 帝王术 / 权术 / 统御 | 政治学 | 中国政治思想 |
| 斯多葛 / 处世哲学 | 哲学 | 伦理学 |
| 声乐 / 唱歌技巧 | 音乐 | 声乐 |
| 手机摄影 / 修图 | 美术学 | 摄影 |
| 情绪调节 / 正念 / 行为改变 | 心理学 | 应用心理学 |

## 给模型的归类指令（生成 / 迁移通用）

> 按中国《研究生教育学科专业目录》体系，为每张卡判定学科归属：
> `deck` 填**一级学科的通行名称**（如金融学、心理学、数学、音乐、哲学、政治学），`category` 填其下的**二级学科或具体细分方向**（中文、单名、不带 `/`）。
> 内容若为实操、技巧、民间或跨界主题，归入**最接近的学术上位学科**（参照上表），不要新造「XX 学」式伪学科，也不要堆斜杠层级。
> 同一 deck 下的 category 应是该学科内部彼此区分的子方向。
